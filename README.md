#RollingDice-App
The RollingDice-App is a simple and fun web application that allows users to roll virtual dice. The app simulates rolling a dice and displays the results on the screen.

The application is designed to be easy to use and can be accessed through any web browser. Simply load the app, click the 'Roll' button, and the app will generate one random numbers between 1 and 6, which will represent the value of die. The value of dice will also be displayed on the screen.

The RollingDice-App is perfect for anyone who wants to play a quick game of dice without having to physically roll the dice. It can be used for a variety of games, such as board games, card games, or even as a fun way to make decisions.

The app is written in HTML, CSS, and JavaScript, and is responsive, meaning it can be used on desktops, laptops, tablets, and mobile devices. It does not require any additional software or plugins to run.

Enjoy rolling the dice and have fun with RollingDice-App!
